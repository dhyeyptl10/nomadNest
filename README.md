## 🌟 What is NomadNest?

</div>

**NomadNest** is a modern travel planning web application that helps explorers discover curated destinations worldwide, plan their perfect trips, and manage all their travel adventures — wrapped in a beautifully animated, premium UI experience.

Built from a pixel-perfect **Figma prototype**, NomadNest brings together AI-powered trip planning, exclusive member deals, and a sleek dashboard — all in one nest for every nomad. 🏕️

## ✨ Features

<details>
<summary><b>🔐 Authentication System</b></summary>
<br/>

- ✅ **Split-screen Login Page** — Mountain hero panel (left) + Auth card (right)
- ✅ **Tabbed Sign In / Create Account** — Smooth crossfade transition between forms

- ✅ **Hero Banner** — Full-bleed travel photo with large typographic headline

- ✅ **Floating Destination Photos** — Two polaroid-style images overlaid on hero
- ✅ **Info Row** — Current Location · Your Next Trip · Upcoming Trip Countdown
- ✅ **Recommended For You** — 4 destination cards with heart toggle + star ratings
- ✅ **Plan Your Next Adventure** — 3-column grid: photos + AI trip planner CTA

- ✅ **Referral Card** — "Get Extra 10% Off" dark promo card in sidebar
- ✅ **Bell Notification** with animated yellow dot indicator

## 🛠️ Tech Stack

</div>

<div align="center">

| Category | Technology | Why |
|:---:|:---:|:---:|
| ⚛️ **Framework** | React 18 | Component-based, fast re-renders |
| ⚡ **Build Tool** | Vite 5 | Blazing fast HMR dev server |
| 🔀 **Routing** | React Router v6 | Client-side page navigation |
| 🎨 **Styling** | Plain CSS + Variables | Zero runtime overhead, full control |
| 🔷 **Icons** | Inline SVG | No external library = smaller bundle |
| 🖋️ **Fonts** | Google Fonts CDN | Playfair Display + DM Sans |
| 🖼️ **Images** | Unsplash CDN | Free high-quality travel photos |

## 📁 Project Structure

</div>

```bash
nomadnest/
│
├── 📄 index.html               # Browser entry — loads React into <div id="root">
├── ⚙️  vite.config.js          # Vite build & dev server configuration
├── 📦 package.json             # Dependencies, scripts, project metadata
│
└── 📂 src/
    │
    ├── 🚀 main.jsx             # ReactDOM.createRoot — bootstraps the entire app
    ├── 🔀 App.jsx              # Route definitions  /→Login  /dashboard→Dashboard
    ├── 🎨 index.css            # Global reset + CSS custom properties
    │
    └── 📂 pages/
        │
        ├── 🔐 LoginPage.jsx    # Auth UI — tabs, form, social login, hero panel
        ├── 🔐 LoginPage.css    # Login animations, split layout, glassmorphism
        │
        ├── 🏠 Dashboard.jsx    # Full dashboard — sidebar, hero, cards, sections
        └── 🏠 Dashboard.css    # Dashboard layout, hover effects, spinning badge
```

## 🚀 Getting Started

</div>

### Prerequisites

```bash
node --version   # v18.0.0 or higher
npm --version    # v9.0.0 or higher
```

### Run Locally

```bash
# Step 1 — Clone the repository
git clone https://github.com/yourusername/nomadnest.git

# Step 2 — Navigate into the folder
cd nomadnest

# Step 3 — Install all dependencies
npm install

# Step 4 — Start development server
npm run dev
```

🌐 Open **[http://localhost:5173](http://localhost:5173)** in your browser.

### Other Commands

```bash
npm run build     # Build optimized production bundle → /dist
npm run preview   # Preview production build locally
```
